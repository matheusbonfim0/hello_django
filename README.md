# Hello Django

Projeto didático com rotas para saudação e operações aritméticas.

```bash
python -m venv .venv
python -m pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

Execute os testes com `python manage.py test`. Configure as variáveis mostradas em
`.env.example`; nunca use a chave de desenvolvimento em produção.
